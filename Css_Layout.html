Modelo de caja

El modelo de caja es una parte de CSS que define unas medidas para los elementos de la página web a modo de bloques. Hay tres medidas que son:

  • Padding: El padding es el espacio de relleno requerido por todos los lados de un elemento. El padding contiene el espacio entre el contenido del elemento y su borde ( border ).
  • Border: Border te permite modificar el aspecto de cada uno de los cuatro bordes de la caja de un elemento. Para cada borde se puede establecer su grosor, su color y su estilo.
  • Margin: Margin establece el margen para los cuatro lados. Es una abreviación para evitar tener que establecer cada lado por separado con las otras propiedades de margen. Pueden ser : margin-top , margin-right , margin-bottom y margin-left .
